# 🎬 Netflix Clone

A responsive front-end clone of the Netflix landing page built using **HTML5** and **CSS3**. This project was created as part of my front-end web development learning journey to practice responsive layouts and modern UI design.

> ⚠️ **Disclaimer:** This project is created for educational purposes only. Netflix is a registered trademark of Netflix, Inc. This project is not affiliated with or endorsed by Netflix.

---

## 📌 Project Overview

This project recreates the Netflix landing page with a clean and responsive design. It focuses on improving front-end development skills by replicating a real-world user interface using HTML and CSS.

---

## ✨ Features

- 🎨 Netflix-inspired modern UI
- 📱 Responsive design
- 🧭 Navigation bar
- 🎬 Hero banner section
- 📧 Email subscription section
- 🖥️ TV information section
- ⚡ Clean and simple code structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/shaikh123-SGH/netflix-clone.git
```

2. Open the project folder.

3. Double-click **index.html** or open it in your preferred browser.
